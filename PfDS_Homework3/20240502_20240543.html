Clustering with K-Means

K-Means is one of the most popular "clustering" algorithms, which you perhaps are already familiar from the Data Mining curricular unit. K-means estimates the location of $k$ centroids that it uses to define clusters. A point is considered to be in a particular cluster if it is closer to that cluster's centroid than any other centroid.

The K-means Algorithm consists in an iterative process that is roughly as follows:

  1. Choose value for K (number of clusters);
  2. Initialize the locations of the centroids, which we can do by randomly select K points from your dataset as initial locations of the centroids;
  3. Calculate distance of all other points to each of the K centroids;
  4. Associate each point to the cluster of the closest centroid;
  5. Update the centroid position, by computing the average coordinates of all points associated to each cluster;
  6. Evaluate the average change in the centroids positions, as a measure of convergence (the algorithm is said to have converged to the solution when the positions of the centroids don't change more than a given tolerance threshold);
  7. Repeat steps 3-6 until either the centroids no longer move more than a tolerance threshold or until you repeated these steps at least for specified number of iterations (niter)

The algorithm is thus parametized by the number of clusters (K), the maximum number of iterations (niter), and the tolerance threshold (tol).

    """Function to calculate the Euclidean distance between two points 'p1' and 'p2'

    Args:
        p1 (np.array): First point
        p2 (np.array): Second point

    Returns:
        distance (float): Euclidean distance between the two points
    """
    # Calculate the squared difference between the two points (p1 and p2)
    squared_diff = (p1 - p2) ** 2 

    # Calculate the sum of the squared differences
    sum_squared_diff = np.sum(squared_diff)

    # Calculate the square root of the sum of the squared differences
    distance = np.sqrt(sum_squared_diff)
    
    # # ------------------------------------ [Alternative] ------------------------------------
    # # Calculate the Euclidean distance using a built-in function from scipy
    # # Source: https://docs.scipy.org/doc/scipy/reference/generated/scipy.spatial.distance.euclidean.html
    # distance = sc.spatial.distance.euclidean(p1, p2)

    # Return the distance
    return distance

# Function to update the centroids of the K-means algorithm
def centroid_update(points: np.array, clusters: np.array) -> np.array:
    """Function to update the centroids of the K-means algorithm

    Args:
        points (np.array): Array with the data points
        clusters (np.array): Array with the cluster assignments of the data points

    Returns:
        centroids (np.array): Array with the updated centroids
    """

    # Calculate the number of clusters (K) - Assuming that the clusters are range from 0 to K-1
    K = int(np.max(clusters) + 1)

    # Create an array to store the centroids
    centroids = np.empty((K, points.shape[1]))

    # Loop through each cluster
    for k in range(K):
        # Get the points that belong to the current cluster 
        current_points = points[clusters == k]

        # Calculate the mean of the points in cluster k

        # --------------- [Check if previous step are well implemented] ----------------
        # 1st Case: If there are points in the cluster, we will calculate the mean of the points
        # 2nd Case: If there are no points in the cluster, we will set as nan (Not a Number)
        if len(current_points) > 0:
            # Set centroid to the mean of the cluster
            mean = np.mean(current_points, axis=0)
        else:
            # If previous steps are not well implemented, there are no points in the cluster to calculate the mean from
            mean = np.nan

        # Append the mean to the centroids array
        centroids[k] = mean

    # Return the centroids
    return centroids

# Function to update the points of the K-means algorithm
def points_update(points: np.array, centroids: np.array) -> np.array:
    """Function to update the points of the K-means algorithm

    Args:
        points (np.array): Array with the data points
        centroids (np.array): Array with the centroids

    Returns:
        closest_centroids (np.array): Array with the closest centroids to each point
    """
    # Create an empty array to store the clusters
    closest_centroids = np.empty(len(points))
    
    # Loop through each point
    for i, point in enumerate(points):

        # Calculate the distance between the point and each centroid
        distances = [distance(point, centroid) for centroid in centroids]

        # Find the index of the centroid that is closer to the point and assign it to the closest_centroids array
        closest_centroids[i] = np.argmin(distances)

    # Return the closest centroids to each point as an integer array
    return closest_centroids.astype(int)

# Function to calculate the error of the K-means algorithm
def error(oldcentroids, newcentroids):
    """Function to calculate the error of the K-means algorithm

    Args:
        oldcentroids (np.array): Array with the old centroids
        newcentroids (np.array): Array with the new centroids

    Raises:
        ValueError: If the old and new centroids don't have the same size

    Returns:
        error (float): Average distance between the old and new centroids
    """

    # Avoid errors when the oldcentroids and newcentroids aren't the same size
    if len(oldcentroids) != len(newcentroids):
        raise ValueError("The old and new centroids must have the same size")

    # Calculate the Euclidean distance between the old and new centroids for each centroid
    distances = [distance(oldcentroids[i], newcentroids[i]) for i in range(len(oldcentroids))]

    # Calculate the average distance
    error = np.mean(distances)

    # Return the error
    return error

# Function to implement the K-means algorithm
def kmeans(points: np.array, K: int, niter: int, tol: float):
    """Function to implement the K-means algorithm

    Args:
        points (np.array): Array with the data points
        K (int): Number of clusters to generate.
        niter (int): Number of iterations to run the algorithm.
        tol (float): Tolerance to stop the algorithm.

    Raises:
        ValueError: If there are empty clusters

    Returns:
        clusters (np.array): Array with the cluster assignments of the data points
        centroids (np.array): Array with the centroids
    """

    # Step 1: Initialize centroids by randomly selecting K unique points
    centroids = init(points, K)

    #         Initialize the clusters as an array of the same size as the points
    clusters = np.empty(len(points), dtype=int)

    #         Initialize the error as infinity
    current_error = np.inf

    # Loop through the number of iterations
    for i in range(0, niter):
        
        # Step 2: Update cluster assignments
        new_clusters = points_update(points, centroids)

        # If there are empty clusters, reinitialize the centroids        
        if len(np.unique(new_clusters)) < K:
            print("Empty cluster detected, reinitializing the centroids...")
            centroids = init(points, K)
            continue
        
        # Step 3: Update centroids based on current cluster assignments
        new_centroids = centroid_update(points, new_clusters)

        # Step 4: Calculate the error between the old and new centroids
        new_error = error(centroids, new_centroids)

        # Step 5: Check for convergence (i.e. If the error is below the tolerance)
        if new_error < tol:
            # If the error is below the tolerance, stop the algorithm
            print(f"Converged after {i+1} iterations")
            break

        # Update the clusters
        clusters = new_clusters

        # Update the centroids
        centroids = new_centroids

        # Update the error
        current_error = new_error

    # Return the final clusters and centroids
    return clusters, centroids

# Function to calculate the inertia of the K-means algorithm
def inertia(points: np.array, clusters: np.array, centroids: np.array):
    """Calculate the inertia of the K-means algorithm.

    Args:
        points (np.array): The data points.
        clusters (np.array): The cluster assigned to each point.
        centroids (np.array): The centroids of the clusters.

    Returns:
        number of clusters (int): The number of clusters.
        inertia (float): The inertia of the K-means algorithm.
    """

    # Initialize the inertia
    inertia = 0

    # Loop through each point
    for i in range(len(points)):
        # Calculate the distance between the point and the centroid of the cluster it belongs to
        squared_distance = distance(points[i], centroids[clusters[i]])**2

        # Update the inertia, summing the squared distance
        inertia += squared_distance

    # Calculate the average squared distance
    avg_squared_distance = inertia / len(points)

    # Return the number of clusters and the average squared distance
    return len(np.unique(clusters)), avg_squared_distance
